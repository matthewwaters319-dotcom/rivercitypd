# Portfolio Website

Red-themed portfolio homepage with a banner, now-playing card, and Discord join card. Built with plain HTML, CSS, and JavaScript.

## Customize
- Replace the profile image by adding rdr2i.png next to index.html (or update the path in styles.css).
- Add back-in-the-saddle.mp3 next to index.html to enable the audio player.
- Add assets/axis-logo.png for the Discord logo and assets/back-in-the-saddle.jpg for the album art.
- Update the Discord member count and server name in index.html.
- Adjust the gradient colors in styles.css under :root.

## Run
Open index.html in a browser.
